# Power BI Dashboard for Higher Officials

This repository contains a one-page Power BI dashboard designed for higher officials, providing key insights into sales performance, profitability, and employee contributions.

## Dashboard Overview

The dashboard includes the following visualizations:

- Product Category Slicer
- Total Sales Card Visual
- Sales by Product Category Stacked Bar Chart
- Net Profit Decomposition Tree
- Country Slicer
- Total Sales by Country Stacked Bar Chart
- Employee Slicer
- Total Sales vs. Targets Clustered Column Chart
- Net Profit by Employee Donut Chart

## Usage

1. Clone or download the repository.
2. Open the `.pbix` file in Power BI Desktop.
3. Ensure that your Power BI Desktop is connected to the appropriate data sources.
4. Interact with the slicers and visuals to explore the data and gain insights.
5. Adjust the layout or add additional visuals as needed for your specific requirements.
6. Save and publish the dashboard to your Power BI service or export it as a PDF for sharing.
